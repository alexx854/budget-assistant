# Описание проекта

Бюджетный помощник - это программа на python для учёта ежедневных расходов пользователя за месяц.
Программа может добавлять расходы по дням, указывать сумму и категорию, считать сумму расходов за выбранный период, находить день с максимальными тратами, сортировать категории по сумме расходов, отменять последнее добавление и отображать расходы в виде дерева

# Возможности проекта

Программа позволяет:
- добавлять расход с указанием дня, суммы и категории
-	считать сумму расходов за период с дня A по день B
-	использовать префиксные суммы для быстрого ответа на интервальные запросы
-	находить день с максимальным расходом с помощью линейного поиска
-	сортировать категории по сумме трат с помощью сортировки вставками
-	отменять последний добавленный расход с помощью стека
-	хранить структуру расходов в виде дерева
-	выводить все расходы за месяц
-	просматривать дерево трат

# Используемые структуры данных и алгоритмы

В проекте используются:
1. Список расходов по дням
- Для хранения расходов за каждый день месяца используется список из 31 элемента. Каждый элемент списка соответствует одному дню месяца
2. Префиксные суммы
- Для быстрого подсчёта суммы расходов за период используется массив префиксных сумм. После каждого добавления или отмены расхода массив префиксных сумм обновляется. Благодаря этому сумма расходов за период считается за O(1)
3. Линейный поиск
- Для поиска дня с максимальным расходом используется линейный поиск. Программа проходит по списку расходов за все дни и находит максимальное значение
4. Сортировка вставками
- Для сортировки категорий по сумме расходов используется сортировка вставками. Категории выводятся по убыванию суммы трат
5. Стек
- Для отмены последнего добавленного расхода используется стек. Последний добавленный расход отменяется первым, то есть используется принцип LIFO
6. Дерево
- Для хранения структуры расходов используется дерево. Это позволяет наглядно отображать расходы по дням и категориям
